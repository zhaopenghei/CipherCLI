# 🔐 CipherCLI-Java

一个简单的 Java 命令行加密工具，使用 AES 对称加密算法加密和解密文本。

## ✨ 功能

- 使用密码进行对称加密（AES）
- 命令行加密和解密支持
- 输出结果为 Base64 编码

## 🚀 使用方法

### 编译项目

```bash
javac CipherCLI.java
```

### 加密文本

```bash
java CipherCLI encrypt "要加密的内容" 你的密码
```

### 解密文本

```bash
java CipherCLI decrypt "加密后的Base64内容" 你的密码
```

## 🛡️ 安全提示

- 使用 SHA-256 生成密钥
- 使用 AES-CBC 模式加密，IV 为默认填充的 0
